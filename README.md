# Rick21 Portfolio

A standalone, minimalist portfolio site inspired by ChatGPT/Claude design language.

## Sections

- Profile — intro, current work, and quick stats
- Projects — full project timeline and roadmap
- Business — reserved for services and partnerships
- Favorite Projects — curated highlights

## Local preview

```bash
python3 -m http.server 8080
```

Then open http://localhost:8080
